001 : PDF Watermarking Project

This project uses the PyPDF2 library in Python to add a watermark to a PDF document. It takes two PDF files as input: the original document and a watermark file. The watermark is applied to each page of the original document, creating a new watermarked PDF output.

"Watermarking PDFs ensures the integrity of our organization's legacy in a rapidly evolving digital world"
